# 🚀 Placement Platform – AI-Powered Student Career Intelligence System

## 📌 Overview

The **Placement Platform** is an end-to-end machine learning and data-driven web application designed to evaluate student placement readiness and provide intelligent career guidance.

It leverages **Machine Learning, Data Analytics, SQL, and NLP-based resume processing** to simulate a real-world placement advisory system. The platform helps students identify skill gaps, assess employability, and follow a personalized career roadmap aligned with industry requirements.

---

## 🎯 Key Features

* 📄 **Automated Resume Parsing**

  * Extracts and analyzes skills from uploaded resumes (PDF/DOCX)

* 🧠 **Placement Readiness Prediction**

  * ML model evaluates student employability score based on skills and profile data

* 📊 **Skill Gap Analysis Engine**

  * Identifies missing technical and professional skills required for target roles

* 🏢 **Company Eligibility Matching**

  * Maps student profiles with suitable companies based on requirements

* 🗺️ **Personalized Learning Roadmap**

  * Generates structured learning paths for skill improvement

* 📈 **Analytics & Visualization**

  * HR-style dashboards using Power BI and Excel for insights and reporting

---

## 🛠️ Tech Stack

* **Programming Language:** Python 3.10
* **Frontend:** Streamlit
* **Machine Learning:** Scikit-learn, Pandas, NumPy, Joblib
* **Data Processing:** Pandas, SQLite (SQL-based storage)
* **NLP & Text Processing:** pdfplumber, python-docx
* **Visualization:** Power BI, Plotly
* **Database:** SQLite

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/placement-platform.git
cd placement-platform
```

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```
